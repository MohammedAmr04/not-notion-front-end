Notifications (MVP) | 

Goals : Keeping the user constantly informed about everything happening within the platform

Story 1: Activity Notifications

As a User I want to receive real-time notifications for new messages, tasks, and system updates , So that I never miss important updates or discussions 

Acceptance Criteria => 

A notification should appear for the user when a new message is received (New Message Alert).

A notification should appear when a new task is created (New Task Alert) or its status is updated (Task Update Alert).

A notification should appear when a new channel is created (New Channel Alert) or a new team is created (New Team Alert).

A notification should appear when a new member is added to the team (New Member Alert).

A notification should appear when any shared document is modified (Doc Update Alert).

Video Calls / Streaming (Phase 2) | 

Goals : To enable direct audio and video communication between team members 


Story 2: Video Conferencing 

As a User I want to initiate one-on-one or group video calls , So that I can collaborate with my team visually

Acceptance Criteria => 

The user can initiate a video call with one member or a group

The user can share their screen during the call 


Story 3: Recording & Streaming 

As a User I want to record calls or stream live video , So that I can document meetings or broadcast to the whole team

Acceptance Criteria => 

Option to record calls and save them for later review.

Ability to live stream to team members.

====================================================================================

File / Media Sharing (Phase 2) | 

Goal: To manage file sharing and control access permissions.

Story 4: File Sharing

As a User I want to upload and download files within channels , This allows me to share resources with my team

Acceptance Criteria => 

Users can upload files and documents within the channel.

Users can download uploaded files.

Story 5: Access Control

As an Admin I want to define who can upload or download files , This allows me to maintain security and manage storage space

Acceptance Criteria =>

Administrator settings should be available to control upload and download permissions for members.
